# Calculadora de Ganhos e Gastos

## Descrição
Este projeto é uma calculadora simples para controlar **ganhos e gastos**.  
O usuário pode inserir valores de vendas e preços para calcular os ganhos, além de registrar gastos para visualizar o balanço.  
A aplicação é feita em **HTML, CSS e JavaScript** e roda diretamente no navegador.

## Tecnologias
- HTML5
- CSS3
- JavaScript (ES Modules)
